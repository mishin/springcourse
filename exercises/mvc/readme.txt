MVC / REST -harjoitus

Aluksi testaa sovelluksen ajoa antamalla komento mvn jetty:run
Tämän jälkeen avaa selaimella osoite http://localhost:8080
Tuloksena pitäisi olla 404-virhe.

Lisää tarvittavat annotaatiot luokkaan com.houston.mvc.MovieController.
Toteuta metodit luokassa com.houston.mvc.RestClient.
Lisää tarvittavat konfiguraatiot tiedostoon movie-servlet.xml.

Testaa taas osoitetta http://localhost:8080 kun kaikki testit luokissa 
com.houston.mvc.MovieControllerTest ja com.houston.mvc.RestClientTest
ovat vihreällä.
